# Neural Architectures

This directory contains the implementations of various neural operator architectures and neural network models used in the HyperNOs project.

## Contents

- **CNO**: Convolutional Neural Operators.
- **FNO**: Fourier Neural Operators.
- **RationalNN**: Rational Neural Networks.
- **ResNet**: Residual Networks.

Each subdirectory typically contains the specific model definitions and associated logic.
