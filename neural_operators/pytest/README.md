# Unit Tests

This directory contains the unit tests for the project, intended to be executed using `pytest`.

## Test Suite

- **Architectures**: Tests for model correctness (`test_cno.py`, `test_fno.py`, `test_residualnetwork.py`).
- **Components**: Tests for loss functions (`test_loss_fun.py`), datasets (`test_datasets.py`), and utilities (`test_utilities.py`).

Run these tests to verify the integrity of the codebase.
